## รายละเอียดโปรเจกต์
แดชบอร์ดนี้ใช้ข้อมูล **Gapminder** เพื่อแสดงสถิติประชากรและอายุขัยเฉลี่ยทั่วโลก
* **Graph 1:** Scatter Plot แสดงความสัมพันธ์ GDP และอายุขัย
* **Graph 2:** Bar Chart แสดงจำนวนประชากรในแต่ละทวีป
* **Graph 3:** Pie Chart แสดงสัดส่วนประชากรโลก
* **Interactivity:** ใช้ Slider เลือกปีเพื่ออัปเดตกราฟทั้ง 3 พร้อมกัน

## วิธีรันบนเครื่อง
1. ติดตั้ง Library: `pip install -r requirements.txt`
2. รันโปรแกรม: `python app.py`
3. เข้าชมผ่าน Browser: `http://127.0.0.1:8050`
